# 電腦零組件歷史價格 TWComponentPricing
## 設計理念
目前市面上有很多比價系統，可以針對指定商品，找出哪裡購買最划算
例如像APP 遊戲盒子，可以列出switch遊戲的歷史價格，優惠最多的品項
故希望對於電腦相關零組件這個領域，可以開發一個查價系統
## 目標功能
1. 查詢單一品項目前價格/歷史價格
2. 查詢目前優惠最多的品項
3. 可以登入追蹤，設定品項特價通知
